## **Descripción del Laboratorio**

Diseño e implementación de un **osciloscopio y generador de señales de nivel básico** utilizando los módulos ADC y DAC del ESP32, integrando elementos de interacción y pruebas con periféricos adicionales.

---

## **Objetivo General**

Diseñar e implementar un sistema básico que permita:

- Visualizar señales de entrada (función de osciloscopio)
- Generar señales analógicas mediante el DAC
- Integrar periféricos de control (pulsador y potenciómetro)
